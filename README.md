# StartupBridge

StartupBridge is a Flutter-based mobile application that connects ALU students with startup opportunities. The platform allows startups to post internships, projects, and volunteer opportunities while enabling students to explore, apply, and manage their applications.

The app provides separate experiences for startups and students:
- **Students** can browse opportunities, filter roles by category, apply for positions, and manage their profiles and applications.
- **Startups** can create opportunities, review applicants, manage posted roles, and connect with talented students.

Built with Flutter and Firebase, StartupBridge provides a seamless, scalable, and user-friendly platform that bridges the gap between emerging talent and innovative startups.
